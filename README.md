Photo Tweeter
=============
Tweets photos.

## Materials
* Tessel 2
* Webcam

## To use
1. Plug in USB webcam to Tessel 2
1. Clone this repo and `npm install`
1. Make sure Tessel is connected to Wifi (`t2 wifi`)
1. `t2 run index.js` to run

Ta-da! Check @TesselTweet's twitter for your tweeted photo.
